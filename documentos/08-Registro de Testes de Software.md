
# Registro de Testes de Software

Relatório com as evidências dos testes de software realizados na aplicação pela equipe, baseado no plano de testes pré-definido.



|Caso de Teste    | CT-01 - Teste de funcionalidade das páginas de cadastro e dashboard, preenchendo do formulário exibindo erros, cadastro finalizado com sucesso ao apertar o botão "entrar", redirecionamento para a página de dashboard, salvamento dos dados no localStorage, barra lateral e menu de navegação aparecendo ao apertar o botão de menu toggle e novos dados em exibição nos cards ao recarregar a página.  |
|:---|:---|
| Resultados obtidos | O usuário facilmente cria uma conta ao preencher os campos do formulário e é avisado quando algum campo estiver incorreto ou incompleto. Seus dados ficam salvos e sua página instantaneamente é redirecionada para a homepage (dashboard) após criar a conta. A aplicação permite ao usuário ver nos cards dentro do dashboard dados sobre o consumo anual, semanal e mensal da sua água, consumo médio por litros, a porcentagem em relação ao primeiro card e dicas rápidas de economia de água, sempre atualizando para novos dados ao recarregar a página.   |
